1. Consume current `source_root_freshness` evidence from the supervised control-host source freshness path.
2. Treat stale resident code behind `origin/main` as a policy blocker for provider-worker active WIP handling.
3. Trigger auto-restart only when stale source evidence is unambiguous and restart can be bounded and observed.
4. Prove restart success before clearing stale resident code policy state.
