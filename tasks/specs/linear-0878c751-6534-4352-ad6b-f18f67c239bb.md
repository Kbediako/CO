This lane restores that coverage without reopening runtime logic.

## Requirements

1. Preserve the current quoted single-token `exec` argv behavior on `origin/main`.
2. Restore truthful raw source-entrypoint coverage for the five CO-136 regressions.
3. Keep the lane scoped to the command-surface seam and required traceability.
4. Avoid runtime edits and broader hot-suite redesign.
5. If later validation reopens a live runtime blocker, stop and re-scope into the smallest real runtime fix.
