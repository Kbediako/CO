## Summary
- Problem Statement: accepted same-issue child-lane workspaces under `.child-lanes/` remain as local untracked provider artifacts after durable run evidence exists, and `scripts/diff-budget.mjs` currently counts those workspace copies as review diff surface.
- Desired Outcome: `diff-budget` excludes only accepted child-lane workspace copies that satisfy the provider-artifact contract, while keeping normal untracked files and unaccepted child-lane output counted.

## User Request Translation
- Complete `CO-328` by making `diff-budget` distinguish accepted `.child-lanes/` workspace artifacts from real user-authored untracked files.
- Preserve the incident evidence from `CO-321`: durable `provider-linear-child-lane.patch` and acceptance ledger evidence existed, but the local `.child-lanes/...` workspace copy still caused `diff-budget` to fail until it was pruned.
- Keep the fix narrow to diff-budget accounting, focused regression coverage, and provider-worker guidance.

## Intent Checksum
- Protected terms: `diff-budget`, `.child-lanes`, accepted same-issue child-lane, `provider-linear-child-lane.patch`, acceptance ledger, untracked budget surface.

## Acceptance Criteria
1. `scripts/diff-budget.mjs` ignores accepted child-lane workspace artifacts only under the safe provider-artifact contract.
2. Regression coverage proves accepted `.child-lanes/` workspace copies do not inflate budget once durable patch/ledger evidence exists.
3. Regression coverage proves unrelated untracked files are still counted.
4. Provider-worker or docs guidance records the lifecycle split between ephemeral `.child-lanes/` workspace copies and durable `.runs/.../provider-linear-child-lane.patch` evidence.
5. The `CO-321` incident evidence is referenced in the implementation packet.

## Non-Goals
- Do not disable `diff-budget`.
- Do not ignore all untracked files.
- Do not ignore child-lane output before durable acceptance evidence exists.
- Do not weaken diff-budget for normal source, docs, test, or config changes.

## Not Done If
- `diff-budget` can still fail on an accepted child-lane `.child-lanes` workspace copy after durable patch/ledger evidence exists.
- A pending or unaccepted `.child-lanes/` workspace is silently excluded.
- Unrelated untracked files stop counting toward the budget.
