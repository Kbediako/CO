# Release Notes Addendum — Shipped Skills

When a release adds or updates bundled skills, include the shipped-skills highlight under **Overview** in the release notes. The release workflow promotes generated **Overview** and **Bug Fixes** into top-level release sections; **Documentation** remains in the generated **Full Changelog** and is not promoted as a shipped-skills highlight section.

Example:
- Shipped new bundled skills: `collab-evals`, `collab-deliberation`, `delegate-early`.
- Install/refresh command: `codex-orchestrator skills install --force`.
- Include the docs link: `docs/skills-release.md`.
